# 💼 Ishita Kumari - Portfolio Website

A modern, responsive full-stack portfolio featuring a pink-themed design, animated canvas background, and real-time contact storage.

[![Live Demo](https://img.shields.io/badge/demo-live-brightgreen)](https://ishita-kumari.netlify.app/)
[![GitHub](https://img.shields.io/badge/github-portfolio-blue)](https://github.com/ishitaray14/portfolio_ishita)

- **Admin Dashboard:** [admin.html](https://ishita-kumari.netlify.app/admin.html)
- **Backend API:** [ishita-portfolio-backend](https://ishita-portfolio-backend.onrender.com)

## ✨ Key Features
- 🎨 **Custom UI:** Beautiful pink theme with animated Canvas bubbles and sparkles.
- 📱 **Fully Responsive:** Optimized for mobile, tablet, and desktop devices.
- 📧 **Full-Stack Form:** Real-time contact submissions stored in a database.
- 📊 **Admin Dashboard:** Interface to view messages with automatic 10s refresh.
- ⏱️ **Tracking:** Automatic timestamping for all incoming inquiries.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6+), Canvas API.
- **Backend:** Node.js, JSON Server, CORS.
- **Deployment:** Netlify (Frontend), Render (Backend), Git.

## 📂 Project Structure
```bash
portfolio_ishita/
├── index.html      # Main portfolio page
├── admin.html      # Message dashboard
├── server.js       # Node.js + JSON Server
├── db.json         # JSON Database
└── package.json    # Dependencies & scripts
```

## ⚡ Quick Start

1. **Clone & Install**
   ```bash
   git clone [https://github.com/ishitaray14/portfolio_ishita.git](https://github.com/ishitaray14/portfolio_ishita.git)
   cd portfolio_ishita
   npm install
2. **Run Backend**

```bash
npm start
Note: Ensure you have Node.js installed. The server will run at http://localhost:3000.
```
